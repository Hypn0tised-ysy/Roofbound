# 移动控制 
## 坐标系
默认角色生成朝向z轴正方向

## 速度
wasd/sprint/jump产生relative speed，相对于平台
角色的实际速度为relative speed与平台速度之和
若角色未移动（没有按下wasd）则继承平台速度（即relative speed为0）
在空中则继承角色离开平台时的速度（即relative speed不变）


## 按键
wasd移动
空格跳跃
shift冲刺
移动鼠标控制视角

up方向(0,1,0),与重力方向相反
鼠标视角正前方:forward
右方向：right=forward×up(叉乘)
w对应的移动方向:moveDir=up×right(叉乘)

## 冲刺
冲刺仅在前进时生效
冲刺时速度为speed*sprintSpeedMultiplier
冲刺持续时间为sprintDuration
冲刺冷却为sprintCooldown
冲刺冷却期间无法冲刺
在空中时无法冲刺(后续添加技能机制)

## 跳跃
跳跃时速度为jumpSpeed
每次落到地面刷新跳跃canJump=true
每次按下空格跳跃，canJump=false(后续用状态机改写)


# 实现
使用unity input system 
角色用character controller组件控制移动
卡车用rigidbody组件控制移动
参数放在SO中
